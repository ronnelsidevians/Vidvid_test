# СПФ Планувальник PWA

- Прибрано видимі версії в інтерфейсі.
- Вибір дільниці/працівника через кнопку «Обрати» поруч із полем та пошукове модальне вікно.
- У полі дільниці показується тільки тип і номер, без тижня.
- У плануванні підказки показують дозволено/заборонено, останнє відвідування і дату доступу.
