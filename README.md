# СПФ Планувальник PWA v12

## Що нового
- Видимий бейдж v12 у шапці.
- Кнопка «Оновити застосунок» очищає service worker/cache.
- Service worker має новий кеш `spf-pwa-v12-20260610` і видаляє старі кеші.
- `app.js?v=12`, `index.html?v=12` для обходу кешу GitHub Pages/PWA.

## Розгортання
Завантажте всі файли в корінь GitHub Pages. Після оновлення відкрийте сайт з `?v=12` або натисніть «Оновити застосунок».
